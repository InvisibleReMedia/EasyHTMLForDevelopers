Welcome

Thanks to have downloaded this software.

Download and extract examples into the EasyHTML directory and then select the menu import to import tools from Basic or jQuery in your own project.

In a project, elements are

  1. Configuration
    You can configure keywords to add them into your pages.
    Each keyword must be present as # (keyword).
  2. Url Javascript
    Add external links to javascript source into your pages.
  3. Url CSS
    Add external links to CSS source into your pages.
  4. Master page
    Create multiple master pages and select your areas to create table and div.
    Add one into a page.
  5. Master object
    Create any master object and select your areas to create table and div.
    Add one into any sub-object in a page, master page or an another master object.
  6. Tool
    Create any tool and write your HTML fragments, Javascript code and specific CSS. Add a copy of your tools into a page, a master page or a master object.
  7. Sculpture
    Create content in one time.
  8. Tool instance
    Contains all copied tools that has been added into your pages
  9. Page
    Add any page and set the path and the file name .html, .php or any extension.
  10. File
    Import files and images.
  11. Folder
    Contains folders
