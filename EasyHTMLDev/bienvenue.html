Bienvenue

Merci d'avoir téléchargé ce logiciel.

Téléchargez et dézippez les exemples dans le répertoire EasyHTML et puis, sélectionnez le menu importer les outils du projet Basic et jQuery dans votre propre projet

Les éléments dans un projet sont :

  1. Configuration
    Vous permet de créer des mots-clés configurables en les incluant dans vos pages.
    Chaque mot-clé doit être presenté comme-ci dans vos pages.
  2. Url de Javascript
    Vous permet d'ajouter des liens externes Javascript pour toutes les pages.
  3. Url de CSS
    Vous permet d'ajouter des liens externes CSS pour toutes les pages.
  4. Modèle de page
    Créez plusieurs modèles de page en sélectionnant les zones pour créer des tables et des div.
    Ajoutez un modèle de page à une page.
  5. Modèle d'objet
    Créez autant de modèle d'objet que vous le souhaitez en sélectionnant les zones pour créer des tables et des div.
    Ajoutez un modèle d'objet dans une page, un modèle de page ou un modèle d'objet.
  6. Outil
    Créez autant d'outils que vous le souhaitez et écrivez vos morceaux HTML, Javascript et CSS spécifiques.
    Ajoutez une copie de vos outils dans une page, un modèle de page ou un modèle d'objet.
  7. Sculpture
    Créez des contenus en une seule fois.
  8. Instance d'outil
    Contient les copies d'outils intégrés dans les pages
  9. Page
    Ajoutez autant de pages que vous le souhaitez, indiquez un chemin physique et un nom de fichier .html, .php ou toute extension.
  10. Fichier
    Incluez des fichiers extérieurs tels que des images.
  11. Dossier
    Contient tous les dossiers.
